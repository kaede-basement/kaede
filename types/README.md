# Types

This directory contains `kaede-lib.d.ts` - a single type declaration file consisting of all Kaede types. With that file, the plugin development will be way easier since it describes every possible field of the `window.__KAEDE__` extension object (`window.__TAURI__`, `window.__TAURI_INTERNALS__`, and `window.__TAURI_PLUGINS_COMMUNITY__` as well).
