# Browser environment

Kaede is intended only to work in Tauri environment. Therefore, in browser environments, Kaede will not work.

...Unless one defines the placeholder functionality of Tauri API. This directory contains the work that is aimed at making Kaede work in Firefox and other browsers.

As for placeholders, instead of an OS filesystem communication, one can use [IndexedDB](https://developer.mozilla.org/en-US/docs/Web/API/IndexedDB_API).
